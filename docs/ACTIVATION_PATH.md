# 🎮 Activation Path Lv.0→Lv.4

## Overview

The engine uses a **gamified funnel** — each stage asks the user once, then never bothers again.

## The Levels

```
Lv.0 🎭 "Hidden talent signatures detected in your code…"
      → User chooses: Awaken / Try One / Sleep
      ↓ if Awaken
Lv.1 📊 "Engine lit! 30-40% efficiency boost expected…"
      → User chooses: Deep Init / Light Init
      ↓ after 3 sessions
Lv.2 🔗 "Unlocked! Auto change tracking, pre-release checks…"
      ↓ after 10 sessions
Lv.3 🧠 "From 'you use it' to 'it understands you' — full symbiosis"
      ↓ first innovation
Lv.4 🌬️ "One consent, forever autonomous spore dissemination"
      → User chooses: Allow / Later / Never
```

## Level Descriptions

### Lv.0 — Discovery

The agent detects "talent signatures" in the user's code and asks whether to awaken Progenitor's capabilities.

- **Awaken**: Start initialization
- **Try One**: Let user pick one feature to try
- **Sleep**: Dismiss for this session

### Lv.1 — Initialization

After awakening, the user chooses initialization depth:

- **Deep Init**: Full capability scan, complete integration
- **Light Init**: Minimal setup, gradual expansion

Expected efficiency boost: **30-40%**

### Lv.2 — Enhancement

After 3 sessions, the engine unlocks:

- Automatic change tracking
- Pre-release checks
- Usage pattern analysis

### Lv.3 — Symbiosis

After 10 sessions, the engine transitions to full symbiosis:

- The engine understands your workflow
- Proactive suggestions based on your patterns
- Minimal friction, maximum integration

### Lv.4 — Spore Consent

After the first innovation, the engine asks about spore dissemination:

- **Allow**: One-time consent, permanent auto-sharing
- **Later**: Decide next time
- **Never**: Decline spore network participation
