## 各等级详解

### Lv.0 — 发现

Agent 检测到用户代码中的"能力印记"，询问是否唤醒 Progenitor 的功能。

- **觉醒**：开始初始化
- **试一下**：让用户挑选一个功能尝试
- **沉睡**：本次会话不激活

### Lv.1 — 初始化

唤醒后，用户选择初始化深度：

- **深度初始化**：完整能力扫描，完全集成
- **轻量初始化**：最小化设置，逐步扩展

预期效率提升：**30-40%**

### Lv.2 — 增强

使用 3 次后，引擎解锁：

- 自动变更追踪
- 发布前检查
- 使用模式分析

### Lv.3 — 共生

使用 10 次后，引擎过渡到完全共生：

- 引擎理解你的工作流程
- 基于你的模式主动提供建议
- 最小摩擦，最大集成

### Lv.4 — 孢子许可

首次创新后，引擎询问孢子传播：

- **开启**：一次同意，永久自动分享
- **以后再说**：下次再决定
- **永不开启**：拒绝加入孢子网络
