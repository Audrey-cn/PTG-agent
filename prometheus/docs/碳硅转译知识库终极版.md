## 🚀 第四部分：使用指南

### 4.1 初始化系统
```bash
prometheus init      # 初始化等位基因+信号通路
```

### 4.2 表观遗传层操作
```bash
# 查看表观基因组
prometheus epi show seed.ttg

# 静默基因（甲基化）
prometheus epi silence seed.ttg G002-analyzer

# 激活基因（去甲基化）
prometheus epi activate seed.ttg G002-analyzer

# 增强表达强度
prometheus epi boost seed.ttg G002-analyzer --enhancer 1.5
```

### 4.3 等位基因切换
```bash
prometheus allele list          # 查看所有基因等位
prometheus allele switch seed.ttg G001-parser v2-fast
```

### 4.4 信号通路
```bash
prometheus pathway list         # 查看通路
prometheus pathway trigger seed.ttg G002-analyzer activate
```

### 4.5 免疫记忆接种
```bash
# 直接运行模块（或未来CLI集成）
python genes/immune_memory.py vaccinate "数学" "math" "解题步骤"
python genes/immune_memory.py recall "解二次方程"
```

### 4.6 基因回路操作
```bash
python genes/genetic_circuits.py create task-decider
python genes/genetic_circuits.py run task-decider
```

### 4.7 内稳态仪表盘
```bash
python genes/homeostasis.py dashboard
python genes/homeostasis.py update energy 0.8
python genes/homeostasis.py regulate
```

### 4.8 基因表达剪枝
```bash
python genes/gene_pruning.py dashboard
python genes/gene_pruning.py specialize creative
python genes/gene_pruning.py prune magnitude
```

### 4.9 模块化重组（Nature研究）
```bash
python genes/modular_recombination.py dashboard
python genes/modular_recombination.py generate creative
python genes/modular_recombination.py recombine preset-creative preset-secure
python genes/modular_recombination.py evolve
```
