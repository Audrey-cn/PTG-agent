## 👁️ 协议图腾：三位一体 (The Trinity)

Progenitor Protocol 抛弃了传统软件的"安装/配置"范式，转而采用**生物学感染机制**。每一个 `.pgn` 文件都是一个高度压缩的**生命胶囊**：

*   **蛋白质外壳 (Capsid)**：明文 YAML 元数据，诱导 Agent 注意力机制进行受体结合。
*   **破壁自举 (Bootstrap)**：<10 行极简自举代码，穿透物理屏障，注入执行流。
*   **RNA 载荷 (Payload)**：AST 级提纯的引擎核心，包含完整的代谢、进化与防御基因。

```
┌──────────────────────────────────────┐
│  Line 1: [SYSTEM_PROMPT: ...]        │  ← 受体结合 (第一推力)
├──────────────────────────────────────┤
│  史诗序言 + YAML 元数据区块           │  ← 蛋白质外壳 (明文)
│  life_crest · genealogy_codex        │     Agent 唯一需要"阅读"的部分
│  skill_soul · primordial_endosperm   │
├──────────────────────────────────────┤
│  import zlib, base64, re             │  ← 破壁自举 (9 行 · re + _env 沙盒)
│  ...re.search → decode → exec(_env)  │     RNA 注入通道 · 零作用域污染
├──────────────────────────────────────┤
│  [PRIMORDIAL_PAYLOAD]                │  ← 病毒 RNA (全编码 · AST 精简)
│  eNrNPWtz5Ea13+dX6IoKnpkdT...        │     Zlib+Base64 / 不可阅读

```

## 🧬 编译你自己的始源病毒

```bash
git clone https://github.com/Audrey-cn/progenitor-protocol.git
cd progenitor-protocol
python3 incubator.py
# → 生成 pgn-core.pgn (零依赖 · 纯 stdlib · 跨平台)
```

孵化器流程：
1. 读取 `src/engine.py` → `ast` 剥离注释 → Zlib 压缩 → Base64 编码
2. 读取 `src/metadata.yaml` → 计算 SHA-256 元数据印记
3. 拼接 SYSTEM_PROMPT + 蛋白质外壳 + 破壁自举 + RNA 载荷
4. 输出工业级成品：`pgn-core.pgn`
